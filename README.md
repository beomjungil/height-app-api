## Getting Started



## Install

If you use npm:

```bash
npm install height-app-api
```

If you use yarn:

```bash
yarn add height-app-api
```

If you use pnpm:

```bash
pnpm add height-app-api
```

It's contains type itself, so you don't need to install `@types/height-app-api`.
